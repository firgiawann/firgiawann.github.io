Makassar, 09 April 2026

Kepada Yth.

Panitia Penyelenggara Seleksi
Beasiswa Bank Indonesia 2026

Perkenalkan, nama saya Firgiawan Listianto, mahasiswa semester empat program studi Pendidikan Teknik Informatika dan Komputer (PTIK) di Universitas Negeri Makassar. Melalui surat ini, saya bermaksud mengajukan permohonan Beasiswa Bank Indonesia sebagai upaya untuk mandiri secara finansial, meringankan beban ekonomi keluarga dalam membiayai studi saya, serta memperluas kontribusi sosial melalui keahlian teknologi yang saya miliki.

Sebagai lulusan SMA jurusan MIPA, masa awal perkuliahan di PTIK merupakan fase transisi yang menantang. Saya harus berupaya ekstra untuk mengejar ketertinggalan dasar teknis dari rekan-rekan lulusan SMK, mulai dari pemahaman struktur komputer hingga logika pemrograman. Namun, tantangan tersebut saya jawab dengan menjadi pembelajar cepat (fast learner). Saya mendalami basis data SQL dan pemrograman web secara mandiri sejak masa pandemi, hingga kini mulai mengeksplorasi konsep Kecerdasan Buatan (AI). Bagi saya, teknologi bukan sekadar baris kode, melainkan solusi praktis untuk mempermudah aktivitas masyarakat.

Bukti nyata kegigihan saya ada pada semester ini, saat saya menyelesaikan tugas besar kolaborasi antara mata kuliah Rekayasa Perangkat Lunak (RPL) dan Pemrograman Mobile. Di tengah keterbatasan perangkat dengan spesifikasi rendah (laptop RAM 4GB), saya tetap berkomitmen membangun aplikasi mobile hingga fungsional. Seluruh progres tersebut saya dokumentasikan melalui platform GitHub (firgiawann) sebagai portofolio teknis saya. Semangat pantang menyerah ini juga saya tuangkan dalam penyusunan proposal Program Kreativitas Mahasiswa (PKM). Meski awalnya merupakan tugas mata kuliah yang diajukan ke tahap seleksi internal, proses tersebut memberi saya pelajaran berharga tentang manajemen proyek yang sistematis.

Sisi pengembangan diri juga saya asah melalui pengalaman kepanitiaan pertama saya di Dies Natalis JTIK (ICE FEST 2025). Sebagai bagian dari divisi Publikasi & Humas, saya memilih untuk memaksa diri keluar dari zona nyaman dengan mengelola komunikasi publik dan berinteraksi langsung bersama banyak orang. Langkah ini secara signifikan meningkatkan kepercayaan diri serta kemampuan manajemen waktu saya di tengah padatnya tugas akademik. Selain itu, keaktifan saya di komunitas GDGoC UNM dan program English Camp 2025 oleh HIMATIK FT-UNM semakin memperkuat kapasitas saya dalam berkolaborasi di lingkungan tim yang dinamis.

Besar harapan saya untuk dapat bergabung dalam komunitas Generasi Baru Indonesia (GenBI). Saya siap mendedikasikan kemampuan teknis dan pengalaman lapangan saya sebagai "Agent of Change" untuk mendukung program literasi digital Bank Indonesia. Saya ingin membuktikan bahwa mahasiswa dengan kemauan belajar yang tinggi dapat memberikan dampak yang relevan bagi masyarakat jika diberikan wadah yang tepat. Terima kasih atas pertimbangan Bapak/Ibu.

Hormat saya,
